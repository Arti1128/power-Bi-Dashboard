# power-Bi-Dashboard
Built an interactive sales dashboard in Power BI to visualize KPIs, sales trends, profit analysis, shipping performance, payment methods, and product category insights. Implemented DAX measures, data modeling, and interactive filters to create a dynamic dashboard that supports business decision-making.

# Key Features:
📈 Sales, Orders, Profit, and Shipping KPI cards
📅 Monthly Sales and Profit trend analysis
🚚 Sales analysis by Ship Mode
🛍️ Sales breakdown by Segment and Sub-Category
💳 Payment Mode analysis
🏢 Regional sales filtering with interactive slicers
🎨 Clean and user-friendly dashboard design
